# Matrix Password Vault

A secure, local password manager built with Python.
Features a "Matrix" inspired green/black UI using CustomTkinter.

## Features
- **Security:** Uses AES encryption (Fernet) and SHA-256 Hashing with Salt.
- **UI:** Modern Dark/Green interface.
- **Local Storage:** Saves encrypted data to a JSON file.

## How to Run
1. Install requirements:
   `pip install -r requirements.txt`
2. Run the app:
   `python matrix_vault.py`
